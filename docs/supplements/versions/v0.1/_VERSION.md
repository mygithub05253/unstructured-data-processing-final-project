## 이 버전에 포함된 산출물

| 파일 | 목적 | 대상 독자 |
|---|---|---|
| `ANALYSIS_프로젝트분석.html` | 교수님 가이드 4개 md 종합 흐름 + 비정형데이터/데이터 마이닝 방법론(TF-IDF, fastText, cosine, 회귀) 시각화 | 비전공자 팀원 + 본인 복습용 |
| `ROLES_역할분담재설계.html` | 4명 모두가 전 단계에 참여하는 협업 모델 재설계 + 개인 폴더 병렬 작업 평가 + 3-Phase 권장 절충안 | 팀 4명 공통 |

## 버전 관리 규칙

코드를 수정하거나 변경 사항이 발생할 때마다 **상위 폴더(versions/)에 다음 버전 폴더(v0.2, v0.3, ...)**를 생성해 백업합니다.

```
docs/supplements/versions/
├── v0.1/                # 현재 (분석+역할분담 초안)
│   ├── _VERSION.md
│   ├── ANALYSIS_프로젝트분석.html
│   └── ROLES_역할분담재설계.html
├── v0.2/                # 다음 버전 (수정 발생 시)
└── ...
```
